# 구글 및 네이버 검색 엔진 사이트 등록 가이드 (Search Engine Registration Guide)

이 가이드는 중개사 AI 클럽 웹사이트를 구글(Google)과 네이버(Naver) 검색 포털에 등록하여 검색 결과 노출을 극대화하고, 생성형 AI 검색(Google AI Overviews, Perplexity 등)의 원천 소스로 원활히 인용되도록 연동하는 단계별 매뉴얼입니다.

---

## 1. 구글 서치 콘솔 (Google Search Console) 등록 프로세스

구글의 크롤링 봇(Googlebot 및 Google-Extended)이 웹사이트를 탐색하고 색인화할 수 있도록 연동합니다.

### 1단계: 서치 콘솔 접속 및 속성 시작
1. [구글 서치 콘솔 공식 홈페이지](https://search.google.com/search-console)에 접속하여 로그인합니다.
2. 속성 선택 창에서 **[URL 접두사]** 옵션을 선택합니다.
3. 입력란에 배포된 도메인 주소(예: `https://realtor-aiclub.vercel.app`)를 프로토콜(`https://`)을 포함하여 입력하고 **[계속]**을 클릭합니다.

### 2단계: 사이트 소유권 확인 (인증)
구글은 도메인의 소유권을 확인하기 위해 두 가지 방식 중 하나를 권장합니다.
- **방법 A: HTML 파일 업로드 (가장 권장)**
  1. 화면에 제공된 구글 전용 인증 파일(예: `google-site-verification.html` 형식)을 다운로드합니다.
  2. 다운로드한 파일을 프로젝트의 `public/` 폴더에 그대로 복사하여 배포합니다.
  3. 배포 완료 후 구글 서치 콘솔 화면에서 **[확인]** 버튼을 클릭하여 완료합니다.
- **방법 B: HTML 메타 태그 추가**
  1. 제공되는 HTML 메타 태그(예: `<meta name="google-site-verification" content="..." />`)를 복사합니다.
  2. 프로젝트의 `index.html` 내 `<head>` 태그 영역에 삽입하여 빌드 배포합니다.
  3. 배포 적용 확인 후 서치 콘솔 창에서 **[확인]**을 누릅니다.

### 3단계: 사이트맵 (Sitemap) 제출
1. 소유권 확인 후 속성 대시보드로 이동합니다.
2. 왼쪽 사이드바 메뉴에서 **[색인]** -> **[Sitemaps]**를 선택합니다.
3. **[새 Sitemap 추가]** 입력 필드에 `sitemap.xml`을 입력하고 **[제출]**을 클릭합니다.
4. 제출 후 상태 항목에 녹색으로 **[성공]** 표시가 나타나는지 확인합니다.

## 2. 네이버 서치어드바이저 (Naver Search Advisor) 등록 프로세스

네이버 검색 엔진 및 AI 서비스와의 색인 동기화를 활성화하기 위한 연동 매뉴얼입니다.

### 1단계: 서치어드바이저 사이트 연동
1. [네이버 서치어드바이저 공식 홈페이지](https://searchadvisor.naver.com/)에 접속해 로그인합니다.
2. 오른쪽 상단의 **[웹마스터 도구]** 버튼을 클릭합니다.
3. **[연동 사이트 목록]**의 주소 입력란에 홈페이지 도메인 주소(예: `https://realtor-aiclub.vercel.app`)를 입력하고 엔터를 입력합니다.

### 2단계: 사이트 소유권 확인
네이버 봇이 소유권을 증명할 수 있도록 인증 설정을 진행합니다.
- **방법 A: HTML 파일 업로드 (가장 권장)**
  1. 제공되는 소유확인 파일(예: `naverxxxxxxxx.html` 형식)을 다운로드합니다.
  2. 다운로드받은 파일을 프로젝트의 `public/` 디렉토리에 저장하여 서버에 배포합니다.
  3. 브라우저로 `사이트주소/naverxxxxxxxx.html`이 정상 노출되는지 테스트합니다.
  4. 웹마스터 도구 화면에서 **[소유확인]** 버튼을 클릭하고 자동등록방지 보안 코드를 입력하여 승인받습니다.
- **방법 B: HTML 태그 주입**
  1. 녹색으로 표기된 메타 태그(예: `<meta name="naver-site-verification" content="..." />`)를 복사합니다.
  2. 프로젝트의 `index.html` 헤더 안에 태그를 삽입하고 배포합니다.
  3. **[소유확인]** 버튼을 통해 완료합니다.

### 3단계: 사이트맵 제출 및 수집 요청
1. 등록된 연동 사이트 목록에서 해당 주소를 클릭하여 상세 리포트 페이지로 이동합니다.
2. 왼쪽 메뉴바에서 **[요청]** -> **[사이트맵 제출]**을 클릭합니다.
3. 입력 필드에 `sitemap.xml`을 타이핑하고 **[확인]**을 누릅니다.

### 4단계: robots.txt 검증
1. 왼쪽 메뉴바에서 **[검증]** -> **[robots.txt]**를 클릭합니다.
2. 하단의 **[수집요청]** 버튼을 눌러, 앞서 설정해둔 robots.txt 설정 파일 규격을 정상적으로 파싱(Parsing)해 가는지 최종 확인합니다.
